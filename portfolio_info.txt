Role: Full Stack & AI Engineer (Architecture + AI Integration)

SupplyBox is a large-scale AI-driven campus recruitment platform designed to connect universities, students, and employers through intelligent automation. The system consists of two major components: a Next.js main platform (werecruit-main) and a FastAPI-based RAG microservice (resume-rag), working together seamlessly to deliver a smooth, data-driven hiring experience.

The platform uses Retrieval-Augmented Generation (RAG) and AI-driven semantic search to help HR teams and universities find the right candidates instantly, process thousands of resumes, conduct automated interviews, and gain deep insights into hiring trends.

‚öôÔ∏è Architecture Overview

SupplyBox follows a two-service architecture:

Frontend (werecruit-main): A full-featured Next.js 14.2.7 web application using TypeScript, Tailwind CSS, and Prisma ORM with PostgreSQL. It handles user dashboards, campaign management, authentication, analytics, and report generation.

Backend (resume-rag): A FastAPI microservice responsible for all AI functionalities including embeddings, resume indexing, and semantic querying. It uses Cohere embeddings for vectorization and Groq LLM for intelligent query optimization.

Both services communicate through secure REST APIs, enabling modular scaling across Render, Railway, or Vercel environments.

Advanced Rubric System:
Includes pre-built and customizable evaluation rubrics for different roles. The AI automatically scores and explains candidate evaluations using detailed per-criterion feedback.

AI Interview System:
Enables asynchronous voice and video interviews with AI-based scoring, cheating detection, and candidate personality insights.

Campus Analytics & Insights:
Provides data-driven insights into placement performance, skill gaps, and hiring trends across multiple universities.

Multi-Campus Management:
Supports universities with batch management, department-level organization, and multilingual support for global scalability.
